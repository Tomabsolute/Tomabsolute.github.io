Markdown语法基础笔记

Markdown是一种轻量级标记语言，创建于2004年，现在已经成为世界上最流行的标记语言之一。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。

为什么选择Markdown？

Markdown具有以下优点：

  1. 易于学习：语法简单直观，没有复杂的标记。
  2. 专注内容：写作时可以专注于内容而不是格式。
  3. 跨平台：几乎所有的平台都支持Markdown。
  4. 兼容性好：可以轻松转换为HTML、PDF等格式。

基本语法

标题

Markdown中的标题使用#符号表示，一级标题使用一个#，二级标题使用两个#，以此类推，最多支持六级标题。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

强调

Markdown使用星号（*）或下划线（_）作为强调标记。

*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体文本*** 或 ___粗斜体文本___

列表

Markdown支持有序列表和无序列表。

无序列表可以使用星号（*）、加号（+）或减号（-）作为列表标记：

* 项目1
* 项目2
* 项目3

+ 项目1
+ 项目2
+ 项目3

- 项目1
- 项目2
- 项目3

有序列表则使用数字加上点号：

1. 第一项
2. 第二项
3. 第三项

链接

Markdown中创建链接的语法很简单：

[链接文本](URL)

例如：[GitHub](https://github.com)

图片

插入图片的语法与链接类似，只是前面多了一个感叹号：

![替代文本](图片URL)

例如：![Markdown Logo](https://markdown-here.com/img/icon256.png)

引用

在Markdown中，你可以使用>符号来创建引用块：

> 这是一个引用。
> 
> 这是引用的第二段。

代码

行内代码使用反引号（`）包裹：

使用`print("Hello World")`函数可以输出文本。

代码块则使用三个反引号（```）包裹，并可以指定语言：

```python
def hello_world():
    print("Hello, World!")
    
hello_world()
```

高级语法

表格

Markdown中创建表格的语法如下：

| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |

任务列表

你可以创建带有复选框的任务列表：

- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务

脚注

Markdown中添加脚注的语法如下：

这里有一个脚注[^1]。

[^1]: 这是脚注的内容。

总结

Markdown是一种简单而强大的标记语言，掌握它可以帮助我们更高效地记录笔记和创建文档。通过这篇基础笔记，希望能帮助你快速入门Markdown语法，开始享受轻松写作的乐趣。

在日常使用中，我建议先掌握基本语法，然后逐步学习高级功能。实践是最好的学习方式，多写多用，很快就能熟练掌握Markdown。
